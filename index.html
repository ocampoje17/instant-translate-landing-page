APP DỊCH NHANH CHO WINDOWS

Viết nội dung bằng ngôn ngữ khác nhanh hơn chỉ với một phím tắt

App giúp bạn dịch nhanh đoạn văn bản đang chọn sang ngôn ngữ cố định bạn đã đặt sẵn. Rất tiện khi viết bình luận trên Reddit, Twitter, trả lời chat với người nước ngoài, hoặc đăng nội dung bằng ngôn ngữ khác từ tiếng chính bạn đang dùng (ví dụ Tiếng Việt).

Tải app cho Windows (x86/x64)
  • Native app cho Windows nên chạy rất nhẹ
  • Hotkey mặc định: Ctrl + Shift + E
  • Dịch sang ngôn ngữ đích cố định theo cài đặt của bạn

TÍNH NĂNG CHÍNH

Thiết kế để giúp bạn viết ngôn ngữ khác tự tin hơn mỗi ngày

Hotkey toàn cục

Chọn đoạn văn bản ở bất kỳ ứng dụng nào trên Windows, nhấn phím tắt để dịch ngay.

Ngôn ngữ đích cố định

Bạn cài một ngôn ngữ đích, sau đó app luôn dịch về đúng ngôn ngữ đó để dùng liền mạch.

Rất tiện cho đời sống hàng ngày

Dùng để viết bình luận mạng xã hội, trả lời chat, hoặc soạn nội dung ngắn bằng ngôn ngữ nước ngoài.

Chạy nhẹ, mở nhanh

Native app cho Windows nên hoạt động mượt và nhẹ, phù hợp để chạy nền lâu dài.

Bảo mật API key

API key hoàn toàn được lưu trên máy bạn, không lo bị lộ API key.

Lưu dữ liệu trên máy

Bản dịch và lịch sử được lưu hoàn toàn trên máy bạn, không gửi đến bất kỳ server nào khác.

CÁCH DÙNG

3 bước để viết bình luận/chat bằng ngôn ngữ khác

  1. Viết ý tưởng của bạn bằng ngôn ngữ chính (ví dụ: Tiếng Việt) rồi chọn đoạn text.
  2. Nhấn tổ hợp hotkey để dịch sang ngôn ngữ đích cố định đã cài.
  3. Dán bản dịch vào Reddit, Twitter, khung chat hoặc nơi bạn muốn đăng.

FAQ

Câu hỏi thường gặp

App có chạy khi đóng cửa sổ không?

Có. Bạn có thể bật tùy chọn chạy nền để app ẩn xuống system tray và tiếp tục lắng nghe hotkey.

API key có an toàn không?

API key hoàn toàn được lưu trên máy bạn, không lo bị lộ API key.

Dữ liệu lịch sử được lưu ở đâu?

Được lưu hoàn toàn trên máy bạn, không gửi đến bất kỳ server nào khác.
